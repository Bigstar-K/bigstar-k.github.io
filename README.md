# Daesung Kim — Portfolio (GitHub Pages)

## Deploy
1. Create a repo named **bigstar-k.github.io** (or use your existing one).
2. Upload **all files** from this folder to the repo root.
3. GitHub → **Settings → Pages**
   - Build and deployment: **Deploy from a branch**
   - Branch: **main** / **root**
4. Wait 1–3 minutes, then open: https://bigstar-k.github.io/

## Notes
- `.nojekyll` is included to prevent Jekyll from processing folders.
- Images are hotlinked from your existing Wix site.
